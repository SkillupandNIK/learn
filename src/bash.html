Bash Programming Rq : POSIX = ensemble de normes pour shell Pour utiliser bash => exécuter #!/bin/bash 2 types de commandes :
  • internes
    • => routines dans le shell
  • externes
    • => execution de programme distinct
3 fichiers standards :
  • 0 : stdin (clavier de base)
  • 1 : stdout (écran du terminal de base)
  • 2 : stderr (écran aussi)
'>' : Rediriger l'output vers un fichier (ou autre ex: programme) '>>' : Rediriger l'output et ajouter si contenu déjà existant
